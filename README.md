## Getting Started

CD into the directory

```bash
cd pydev-psets
```

Then, build:

```bash
. ./activate
```

Then, run tests:

```bash
invoke test -m pset_basic_data_types
```


Done!

## Dev notes

To watch / build tests:

```bash
$ invoke watch
```
